# Sequential Circuit Design using Verilog

## Intern Alpha VLSI Internship - Task 1

### Objective
To design and simulate various sequential circuits using Verilog HDL and verify their operation using GTKWave.

### Components Implemented

#### Flip-Flops
- SR Flip-Flop
- JK Flip-Flop
- D Flip-Flop
- T Flip-Flop

#### Sequential Circuits
- 4-bit Up Counter
- 4-bit Shift Register
- Traffic Light Controller FSM

### Tools Used
- Verilog HDL
- Icarus Verilog
- GTKWave
- Visual Studio Code

### Results
All modules were successfully simulated and verified through waveform analysis.
